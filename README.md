# Description-of-project
This project is basically a website I have created to display my favourite photos.
It is a portfolio and it has four sections.
The four sections are:Home,Gallery,About,Contact.
The Home section contains a small description of the Webpage.
The Gallery section has some of my favourite photos displayed in grid-like manner.
The About section contains a breif biography including a photo of me.
The Contact section contains a form where you can enter your details so as to contact me.
