# Photography Website

## Author
Johncarlos Mwenda

## Description
This project is a portofolio website I created to display my favourite photos.It is a portofolio with four sections:
- **Home** - Contains a small description ot the webpage
- **Gallery** - Displays my favouritephotos in a grid-like manner
- **About** - Contains a brief biography including a photo of me
- **Contact** - Contains a form where you can enter details to contact me

## Project Setup Instructions
1. Clone the repository from Github
2. Open `index.html` in your web browser
3. Alternatively, visit the live site link below
